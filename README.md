# (Student Grade Generator)
## Description
It is a program that prompts the user to input student marks and generates a grade according to the marks.
The range of marks should be between 0 and 100.
Using this the program categorizes the marks according to the following grade values:
A > 79, B - 60 to 79, C -  49 to 59, D - 40 to 49, E - less 40.

How it works:
1. Prompts the user to input their students marks.
2. It displays the corresponding grade based on student marks


# (Speed Detector)
## Description
It is a program that takes as input the speed of a car e.g 80. 
If the speed is less than 70, it should print “Ok”. 
Otherwise, for every 5 km/s above the speed limit (70), it should give the driver a particualr number of demerit points based on speed.

How it works:
1. Prompts user to input their car speed.
2. It displays whether the car speed is within speed limit and if not shows the accrued demerit points.


# (Net Salary Calculator)
## Description
It is a program that majorly caluclates an individual’s Net Salary by getting the inputs of basic salary and benefits. 
It also considers the following deductions: payee, NHIFDeductions, NSSFDeductions, gross salary, and net salary. 

How it works:
1. Prompts user to input basic salary and benefits values.
2. It the displays the gross salary, paye, NHIF and NSSF deductions along with the net salary.
